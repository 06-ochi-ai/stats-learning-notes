# 不偏分散のシミュレーション(2026年6月)

母分散と不偏分散(標本分散)の違いについて、なぜ不偏分散は `n-1` で割る必要があるのかをPythonでシミュレーションして検証したノートブックです。

## きっかけ

Excelで分散を求める際に「VAR.S」「VAR.P」という2つの関数があり、どちらを使うべきか混乱したことが出発点です。公式として「標本ならVAR.S(不偏分散)」と覚えるだけでなく、なぜn-1で割る必要があるのかを自分の手で確かめたいと思い作成しました。

## 内容

- 母分散と不偏分散の数式の違い($n$で割るか$n-1$で割るか、$\mu$か$\bar{X}$か)
- 標本平均を使うことで分散を小さく見積もってしまう理由
- サンプルサイズ(1〜100)を変化させたときの、2つの分散の挙動比較シミュレーション
- 数式による証明

## 使用ライブラリ

- NumPy

## シミュレーション概要

### 目的

母分散($n$で割る)と不偏分散($n-1$で割る)のどちらが、実際に母集団の真の分散に近い値を返すのかを検証する。

### 手順

1. **母集団の作成**
   正規分布に従う乱数を100,000件生成し、これを母集団とする(母分散は既知の値として固定される)。

2. **サンプリングと分散の計算**
   サンプルサイズを1〜100まで変化させ、各サンプルサイズについて以下を実行する。
   - 母集団から指定サイズのサンプルをランダムに抽出
   - 標本平均を使って、$n$で割った分散($\verb|np.var(sample)|$)を計算
   - 同じサンプルで、$n-1$で割った不偏分散($\verb|np.var(sample, ddof=1)|$)を計算
   - 上記を1,000回繰り返し、平均値を取る(1回のサンプリングでは結果がブレるため)

3. **比較**
   各サンプルサイズにおける2種類の分散の平均値と、母分散との差をグラフに描画する。

### 結果

- サンプルサイズが小さいほど、$n$で割った分散は母分散より小さく出る傾向がある
- $n-1$で割った不偏分散は、サンプルサイズが小さい段階でも母分散に近い値を保つ
- サンプルサイズを増やすにつれて、両者の差は徐々に縮んでいく

→ 母分散を推定する目的であれば、サンプルサイズに関わらず不偏分散($n-1$で割る方)を使う方が適切であることが確認できた。
